# Resumen de Fases (Oleadas de Desarrollo)

El proyecto se ejecutó y auditó siguiendo una hoja de ruta dividida en 4 oleadas estratégicas para mantener el control de calidad:

* **Fase 1: Configuración Global y Presentación.**
    Acondicionamiento del repositorio `PortafolioPersonal` (Edición del *About*, foto de perfil y `README.md` global) para establecer una presencia profesional antes de desplegar código.
* **Fase 2: Despliegue Base y Auditoría Técnica.**
    Empaquetado del proyecto (`npm run build`), configuración de Git para rastrear la carpeta `/dist`, despliegue en GitHub Pages y validación de enlaces.
* **Fase 3: Mejoras, Implementación y Refactorización.**
    Integración de nuevos datos (6ta ruta añadida), cambios en copys ("Modo Romántico") y aplicación de mejoras UI/UX sobre los componentes de React, respetando los requerimientos de la materia DWF.
* **Fase 4: Documentación y Cierre.**
    Auditoría final del código (Criterios de honestidad), captura de evidencias responsivas y redacción de este conjunto de documentos (`docs/`) para avalar el proceso de ingeniería detrás de la interfaz.
