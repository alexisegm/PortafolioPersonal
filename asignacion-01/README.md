# Bitacora-de-entrenamiento
Aplicacion web frontend para el resgistro y seguimiento de rutinas de acondicionamiento físico. Desarrollada con Vanilla JavaScript bajo una arquitectura limpia en 3 capas (Datos, Lógica, UI) para garantizar la escalabilidad y un código modular.  

# Bitácora de Entrenamiento Frontend 🏋️‍♂️💻

## Descripción
Una aplicación web interactiva diseñada para la gestión, registro y seguimiento de rutinas físicas. Este proyecto nace con el objetivo de facilitar el control del entrenamiento diario, estructurando las rutinas de manera clara y accesible.

A nivel técnico, el proyecto está construido puramente con tecnologías web estándar, haciendo énfasis en las buenas prácticas de programación, la modularidad del código y el trabajo colaborativo.

## Arquitectura del Proyecto
El desarrollo se estructuró utilizando una **Arquitectura de 3 Capas** para separar las responsabilidades, facilitando así el mantenimiento y la escalabilidad:
*   **Capa de Datos (`data`):** Manejo de *mock data* para simular la persistencia de las rutinas y ejercicios.
*   **Capa de Lógica (`logic`):** Procesamiento de la información, filtrado de entrenamientos y reglas de negocio.
*   **Capa de Presentación (`ui`):** Manipulación del DOM e interacciones del usuario.

## Tecnologías Utilizadas
*   **HTML5:** Estructura semántica.
*   **CSS3:** Estilos y diseño responsivo.
*   **Vanilla JavaScript:** Lógica de la aplicación sin frameworks externos.
