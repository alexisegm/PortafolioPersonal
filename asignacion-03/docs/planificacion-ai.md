# Planificación y Uso de IA Generativa

Durante el desarrollo de EconoRutas, se integraron herramientas de IA Generativa como asistentes de codificación y estructuración, manteniendo siempre el control humano sobre la lógica de negocio.

## 1. Casos de Uso Específicos
* **Configuración de Entorno:** Generación de instrucciones precisas para la configuración del archivo `vite.config.js` (`base: '/...'`) asegurando un despliegue exitoso en GitHub Pages evitando errores de enrutamiento estático.
* **Refactorización UI/UX:** Asistencia para rastrear y reestructurar selectores complejos dentro del archivo `index.css`, permitiendo aplicar mejoras visuales de forma segura.
* **Estructuración Base:** Apoyo en el diseño del esqueleto inicial de los componentes en React antes de la inyección manual de datos.

## 2. Decisiones Humanas (No delegadas)
* Conceptualización del modelo de EconoRutas y recolección de la data real de las locaciones caraqueñas.
* Implementación manual de las `props` y el pase de parámetros entre componentes padres e hijos.
* Auditoría de calidad y estructuración de la hoja de ruta de despliegue en fases lógicas.
